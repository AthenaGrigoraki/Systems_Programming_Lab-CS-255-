Αθηνά Γρηγοράκη csd4109
1.Με την assert μπορούμε απλά να ελέγξουμε αν είναι null,
δεν μπορούν να εξασφαλίσουν οτι η μνήμη προορισμού είναι αρκετά μεγάλη.
2.Γίνεται με assert(n) να γίνει έλεγχος για το αν η παράμετρος είναι 
αρνητική. Αν όμως χρησιμοποιήσω size_t δεν χρειάζεται καθώς είναι 
τύπος που δεν υποστηρίζει αρνητικές τιμές(unsigned).
